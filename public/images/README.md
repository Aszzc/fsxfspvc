# 图片资源占位说明

本目录用于存放网站的真实图片资源。当前为重建第一版，所有图片使用 CSS / HTML 中的占位样式。

## 待替换文件

将以下真实图片放入 `public/images/` 目录后，网站会自动使用真实图片（无需修改代码）：

| 文件名 | 用途 | 位置 |
| --- | --- | --- |
| `factory-warehouse.jpg` | 工厂仓库实拍（首屏 + 多处） | 首页 hero、产品目录等 |
| `factory-production.jpg` | 生产线实拍 | 制造能力页 |
| `factory-mixing.jpg` | 混料工序 | 制造能力页 |
| `factory-extrusion.jpg` | 挤出造粒工序 | 制造能力页 |
| `factory-storage.jpg` | 成品仓储 | 制造能力页 |
| `wechat-qr.jpg` | 微信二维码（加好友） | 联系页 |

## 图片建议

- 格式：JPG（照片），WebP（可选，体积更小）
- 尺寸：宽度 1200-1600px 足够，移动端会自动适配
- 大小：单张控制在 300KB 以内，首页首图控制在 500KB 以内
- 内容：真实工厂、真实仓库、真实生产线、客户允许展示的实物

## 替换方式

将图片放入 `public/images/` 对应文件名后，重新部署即可。Astro 会自动从 `public/` 目录复制文件到 `dist/`。
